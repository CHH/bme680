# bme680

Node.js module for controlling [Bosch Sensortec BME680](https://ae-bst.resource.bosch.com/media/_tech/media/datasheets/BST-BME680-DS001-00.pdf) sensors.

## Installation

```sh
npm install bme680-sensor
```
